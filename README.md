# Лабораторна робота №3

### Тема
**Багаторівнева архітектура та симуляція БД для "Дуельного Клубу"**

### Мета
Закріпити знання про багаторівневу архітектуру (Entities, Repository, Service, DTOs, Mappers) на основі лекційного матеріалу. Навчитись реалізовувати паттерн "Repository" для інкапсуляції логіки доступу до даних та "Service" для бізнес-логіки. Рефакторинг попередньої лабораторної роботи для роботи з симульованою базою даних.
